# Auditly.ai 🛡️

### **Financial Intelligence for the 2026 AI-Native Workforce**

Auditly AI is a specialized tool designed to combat **"Subscription Drift."** As developers and startups rapidly adopt AI tools, financial leaks occur through redundant subscriptions. Auditly performs a deep audit to recover lost runway.

---

## 🚀 Final Build Status: [Dual-Framework MVP]

**Completion Date:** May 11, 2026

The project is a cross-platform solution, demonstrating the ability to deploy the same business logic across different technical environments.

### 🛠️ Core Features

* **Hybrid Engine:** Framework-agnostic redundancy logic implemented in **TypeScript** (`auditLogic.ts`) and **Python** (`app.py`).
* **Interactive Metrics:** Real-time dashboards for "Current Spend" vs. "Potential Savings."
* **Dynamic Currency Toggle:** Instant conversion between **INR (₹)** and **USD ($)** at 2026 rates ($1 = ₹84$).
* **Social Proof:** Integrated sharing functionality to drive organic user growth.

## 💻 Technical Setup

```bash

pip install streamlit

streamlit run app.py

```

### **Option B: Next.js Version (UX Review)**

Recommended for evaluating frontend performance and UI/UX.

```bash

npm install

# Run the development server
npm run dev
```
