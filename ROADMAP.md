# Product Roadmap: Auditly.ai (Post-May 2026)

## Q3 2026: Automated Discovery
- **Browser Extension:** Identify AI subscriptions by scanning Gmail/Outlook receipt headers locally.
- **Bank API Integration:** Connect via Secure-Read-Only APIs to pull actual "Burn Rates" automatically.

## Q4 2026: Team Intelligence
- **Usage-to-Cost Mapping:** Identify seats that are paid for but haven't made an API call in 30 days.
- **Enterprise SSO:** Single Sign-On for large Indian tech hubs (Bengaluru/Hyderabad).

## 2027: The "AI Purchasing Agent"
- **Auto-Negotiator:** An AI agent that negotiates "Team Tiers" with companies like OpenAI or Anthropic on behalf of the user.
